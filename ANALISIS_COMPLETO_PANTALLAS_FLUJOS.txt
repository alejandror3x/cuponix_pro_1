  cupones de cuentas que sigues.
Tipo: Pantalla completa
Componentes:
  - BBI: Botón Inicio
  - BCI: Botón Centro  
  - BPI: Botón Pie
  - BMsI: Botón Mensajes
  - BOI: Botón Opciones
  - VIR#1, VIR#2, VIR#3: Ventanas Respuesta IA (3 ventanas)
  - PES2: 2 Pestañas
  - VIP1#1: Ventana Información Pop-up
  - BS3#1, BS3#2: Botones Secundarios
  - BS2#1: Botón Secundario
Transiciones:
  → V22 (siguiente)
  → V27 (historial/transacciones)
  → V40(ACT) (actividades)
  → V19(MSJ) (mensajes)
  → V16(X) (cerrar/atrás)
Pestañas: "Para Ti" / "Siguiendo"

V22 - BÚSQUEDA / CATEGORÍAS
---------------------------------------
Descripción: Ventana buscador general. Cuenta con IA (Gemini para búsqueda).
  Se puede buscar por etiqueta/categoría.
Tipo: Pantalla completa
Componentes:
  - Buscador IA integrado
  - Selector de categorías
Transiciones:
  → V23(CUE) (resultados de búsqueda)
  → V21 (atrás a home)
Categorías disponibles: Salud, Tecnología, Maquillaje, Arte, Deporte, Ropa, 
  Restaurante, Pizza, Sushi, Artesanía, Bares, Gym, Vegetariano, Mascotas,
  Música, Cótel, Hamburguesa, Cine

V23(CUE) - RESULTADOS DE BÚSQUEDA / CUPONES A EXPLORAR  
---------------------------------------
Descripción: Resultado en cuentas y cupones de la búsqueda realizadas.
  Muestra cupones recomendados según búsqueda.
Tipo: Pantalla completa (scrolleable)
Componentes:
  - VIR: Ventana Respuesta IA
  - PES2: 2 Pestañas (Cupones / Cuentas)
  - CT1: Contenedor principal
  - BS2#1, BS3#1, BS3#2: Botones secundarios
  - VIP1: Ventana de Información
Transiciones:
  → V24 (ver detalle de cupón)
  → V25 (detalle ampliado)
  → V21 (atrás)
  → V27 (transacciones)
Pestañas: "Cupones Explorar (CUE)" / "Otra pestaña"
UI Elements: 
  - Tarjetas de cupones (330x330px / 165x165px)
  - Descuentos, Recompensas, Fecha de expiración
  - Botón "Guardar" para guardar cupón

Descripción: Ventana de código QR de transacción. Se genera al usar el cupón.
  Este código debe mostrarse al propietario de la cuenta.

Componentes:
  - BHI, BCI, BPI: Botones principales
  - BR: Botón Regresar

  - CT1: Contenedor

  - PES2: 2 Pestañas
  - VIP1: Ventana Información

Transiciones:
  → V36[V25.POP] (popup de guardar)
  → V21 (atrás)
  → V27 (transacciones)
  → V40 (actividades)
Funcionalidades:
  - Display de código QR
  - Información de puntos recompensados
  - Resumen de cupón usado

V36[V25.POP] - MODAL: GUARDAR CUPÓN
---------------------------------------
Descripción: Popup modal que pregunta "¿Deseas guardar este cupón para otro momento?"
Tipo: Modal/Popup
Componentes:
  - BHI, BCI, BPI: Botones
  - BS1: Botón Secundario
  - POP.1, POP.2: 2 Popup elements
Botones:
  - "Guardar" → Guarda el cupón
  - "Ahora no" → Cancela
Transiciones:
  → V25 (volver)
  → V36 (puntos)

12. COMPONENTES UI GLOBALES
================================================================================

Botones Principales (BX):
  - BBI: Botón Inicio/Top
  - BCI: Botón Centro
  - BPI: Botón Pie/Bottom
  - BMsI: Botón Mensajes
  - BOI: Botón Opciones
  - BCrI: Botón Crear
  - BGI: Botón General
  - BHI: Botón Historia
  - BR: Botón Regresar
  - BCA: Botón Centro Arriba

Botones Secundarios (BS):
  - BS1, BS2, BS3: Variaciones de botones secundarios
  - BS#1, BS#2: Múltiples instancias del mismo botón

Contenedores (CT):
  - CT1: Contenedor principal
  - CT2: Contenedor secundario

Ventanas especiales (V):
  - VIR: Ventana de Respuesta IA (3 instancias en V21)
  - VIP: Ventana de Información Pop-up (4 instancias)
  - VIQR: Código QR (360x369px)

Pestañas/Tabs (PES):
  - PES.1, PES.2: Número de pestañas
  - PES2: Específicamente 2 pestañas

Modales/Popups (POP):
  - POP.1, POP.2: Múltiples popups

Entrada de datos (E):
  - E: Campo entrada (máx. 14 o 200 caracteres según contexto)
