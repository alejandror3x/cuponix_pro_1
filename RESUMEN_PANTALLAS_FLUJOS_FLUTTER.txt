TARJETAS CUPÓN:
───────────────
- Tamaño: 330x330px (grande) / 165x165px (pequeña)
- BorderRadius: R3
- Información: Nombre cuenta, descuento, compra mínima, recompensa, expiración
- Acciones: Usar, Guardar
- Componentes: Foto perfil (120x120px), texto de cuenta

CÓDIGO QR:
──────────
- Tamaño: 360x369px
- Tipo: VIQR (ventana QR)
- Uso: Transacciones entre usuarios
- Ubicación: V25, V28

PESTAÑAS (Tabs):
────────────────
- PES2: 2 pestañas (común)
  - V21: "Para Ti" / "Siguiendo"
  - V23: "Cupones" / "Cuentas"
  - V25: Pestañas en detalle
- PES.1, PES.2: Variable según contexto

BOTONES:
────────
- BBI: Top button (historial/info)
- BCI: Center button (principal action)
- BPI: Bottom button (secondary action)
- BMsI: Messages button (siempre visible)
- BOI: Options button (menu)
- BS1/2/3: Secondary buttons (variaciones)
- BR: Back button (regresar)

CAMPOS ENTRADA:
───────────────
- E: Input field
- Máximo 14 caracteres (código cupón)
- Máximo 200 caracteres (búsqueda)
- Búsqueda: Soporta etiquetas con IA

VENTANAS ESPECIALES:
────────────────────
- VIR: Ventana Respuesta IA (660x480px) [En V21: 3 instancias]
- VIP: Ventana Info Popup (4 instancias típicamente)
- CT1/CT2: Contenedores scrolleables

ARQUITECTURA:
─────────────
✓ Usar Navigator 2.0 o go_router para manejo de rutas complejas
✓ Provider/Riverpod para state management
✓ Separar pantallas en widgets reutilizables

WIDGETS RECOMENDADOS:
─────────────────────
✓ CustomScrollView con SliverAppBar para V21, V23
✓ TabBar para pestañas (PES2)
✓ QrImageView para display QR (qr_flutter package)
✓ QrScanner para V28 (qr_scanner package)
✓ BottomNavigationBar o NavigationRail
✓ Material Dialog para modals
✓ GridView para tarjetas de cupones

PACKAGES SUGERIDOS:
───────────────────
- go_router: Navegación avanzada
- provider/riverpod: State management
- http/dio: API calls
- sqflite: Local persistence
- qr_flutter: QR generation
- qr_scanner: QR reading
- firebase: Auth, messaging, notifications
- flutter_gemini: Integración con Gemini API
- cached_network_image: Optimizar imágenes
- shimmer: Loading effects

PERSISTENCIA:
──────────────
- SharedPreferences: Preferencias de usuario
- Sqflite: Historial local
- Hive: Cache de cupones
- Firebase: Datos en cloud

OPTIMIZACIONES:
────────────────
- Lazy loading en listas (V21, V23, V27, V35)
- Caching de búsquedas (V22)
- Preload de imágenes de cupones
- Paginación en historial
